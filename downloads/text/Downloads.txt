\section DownloadsExt External resource files

During setup, \c SKIRT loads built-in SEDs and dust properties from resource data files. Most of these resource files
are provided as part of the source code repository, and thus are automatically available after obtaining the \c SKIRT
code. Some resource files, however, are too large for inclusion in the code repository. Instead, they are offered here
for download. After downloading, extract the contents of the archive and place the resulting files in a newly created
directory called \c SKIRT/extdat next to the \c SKIRT/git and \c SKIRT/release directories.

 - <a href="http://sciences.ugent.be/skirtextdat/PolarizationProperties.tar.gz">PolarizationProperties.tar.gz</a>:
   optical dust properties used by the \c PolarizedSilicateGrainComposition and \c PolarizedGraphiteGrainComposition
   classes
 - <a href="http://sciences.ugent.be/skirtextdat/FSPSVariableIMFSED.tar.gz">FSPSVariableIMFSED.tar.gz</a>:
   template data for the SED families with variable initial mass function used by the \c FSPSVariableIMFSEDFamily class
